## 📋 核心功能快速使用

### 1️⃣ 添加司机（自动创建账号）

1. 登录后点击"录入司机信息"
2. 填写司机基本信息：
   - 工号（必填）
   - 姓名（必填）
   - 性别、身份证、联系方式等
3. 填写驾驶信息：
   - 驾驶证号、类型、有效期
   - 驾龄、所属线路等
4. 点击"提交"

✅ **系统会自动创建用户账号**
- 用户名：司机工号
- 密码：123456
- 角色：司机（driver）

### 2️⃣ 录入违章信息（含视频）

1. 点击"录入违章信息"
2. 选择违章司机、车辆、类型
3. 填写违章时间、地点等信息
4. **上传违章视频**（可选）：
   - 点击"📹 选择视频"
   - 选择视频文件（≤100MB）
   - 等待上传完成
   - 可预览视频
5. 点击"提交"

**支持的视频格式**：
- MP4、AVI、MOV、WMV、FLV、MKV、WebM

### 3️⃣ 查询违章记录（查看视频）

1. 点击"查询司机违章详情"
2. 选择司机和时间范围
3. 点击"查询"
4. 在结果表格中：
   - 有视频的记录显示"📹 查看视频"按钮
   - 点击按钮在弹窗中播放视频
   - 点击关闭或背景退出

## ⚠️ 常见问题

### Q1: 启动失败，提示MySQL未运行
**解决方法**：
```bash
# 以管理员身份运行命令提示符
net start MySQL80
```

### Q2: 视频上传失败
**可能原因**：
1. 文件太大（>100MB）
2. 格式不支持
3. 网络中断

**解决方法**：
- 压缩视频或转换为MP4格式
- 检查网络连接
- 重新上传

### Q3: 视频无法播放
**解决方法**：
1. 确保浏览器支持视频格式
2. 转换为MP4格式（兼容性最好）
3. 检查文件是否完整

### Q4: 新增司机没有出现在用户管理
**解决方法**：
运行补丁脚本：
```bash
python create_missing_driver_accounts.py
```
