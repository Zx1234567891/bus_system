# AI智能助手 - 使用说明

## 🤖 功能介绍

AI智能助手是公交安全管理系统的智能客服功能，由DeepSeek大语言模型驱动，专门为本系统定制训练，能够：

- ✅ 解答系统功能和使用方法
- ✅ 提供操作步骤指导
- ✅ 解决常见问题
- ✅ 介绍系统特性
- ✅ 24/7全天候在线服务

---

## 📍 访问入口

1. 登录系统后
2. 点击左侧导航栏的 **🤖 AI智能助手**
3. 进入聊天界面

---

## 💬 使用方式

### 1. 快速提问
点击预设的建议问题，快速获取答案：
- 📝 如何录入司机信息？
- 🎥 如何上传违章视频？
- 🔍 如何查询司机违章记录？
- 📋 系统有哪些功能？

### 2. 自由提问
在输入框中输入您的问题：
- **发送消息**：按 `Enter` 键
- **换行**：按 `Shift + Enter`
- **清空对话**：点击右上角"🗑️ 清空对话"

### 3. 查看回复
- AI会实时回复您的问题
- 支持Markdown格式（**粗体**、*斜体*、`代码`等）
- 自动滚动到最新消息

## 🎯 示例对话

### 示例1：了解功能
```
用户：系统有哪些功能？
助手：公交安全管理系统主要包括以下功能：
1. 司机管理 - 录入司机信息，自动创建用户账号
2. 车辆管理 - 录入车辆信息
3. 违章记录管理 - 录入违章信息，支持上传视频
4. 查询统计 - 查询司机违章记录、车队统计等
5. 用户管理 - 管理系统用户（仅管理员）
...
```

### 示例2：操作指导
```
用户：如何上传违章视频？
助手：上传违章视频的步骤：
1. 进入"录入违章信息"页面
2. 填写违章基本信息
3. 点击"📹 选择视频"按钮
4. 选择视频文件（最大100MB）
5. 等待上传完成，可预览视频
6. 填写其他信息后提交
...
```

### 示例3：问题解决
```
用户：新增的司机没有在用户管理中显示怎么办？
助手：这个问题可以这样解决：
系统已经支持自动创建账号功能，但如果是之前添加的司机，
可以运行补丁脚本：
`python create_missing_driver_accounts.py`
这会为所有没有账号的司机创建账号。
...
```

## 💡 使用技巧

### 1. 提问技巧
✅ **清晰具体**
- 好：如何录入司机信息？
- 差：怎么用？

✅ **一次一个问题**
- 好：如何上传视频？
- 差：怎么录入违章？怎么上传视频？怎么查询？

✅ **提供上下文**
- 好：我是管理员，如何重置司机密码？
- 差：怎么重置密码？

### 2. 对话管理
- **清空对话**：定期清空对话，避免上下文混乱
- **新话题**：开始新话题时建议清空之前的对话
- **持续对话**：相关问题可以连续提问

### 3. 结果验证
- AI的回答基于系统知识库
- 如有疑问，可以实际操作验证
- 复杂问题建议咨询技术支持

## ⚠️ 注意事项

### 1. 网络要求
- 需要稳定的网络连接
- 需要访问DeepSeek API
- 响应时间受网络影响

### 2. 内容限制
- 仅回答系统相关问题
- 不处理系统外部问题
- 不执行实际操作（仅指导）

### 3. 隐私安全
- 不要在对话中输入敏感信息
- 不要分享密码或密钥
- 对话不会永久保存

### 4. 使用限制
- 每次请求最大2000 tokens回复
- 保留最近10轮对话上下文
- 超时30秒自动终止

## 🔍 常见问题

### Q1: AI助手无响应怎么办？
**原因**：网络问题或API服务异常
**解决**：
1. 检查网络连接
2. 刷新页面重试
3. 稍后再试

### Q2: 回复不准确怎么办？
**解决**：
1. 尝试换个方式提问
2. 提供更多上下文信息
3. 清空对话重新开始

### Q3: 可以用中文提问吗？
**答**：完全支持中文！AI助手专为中文用户优化。

### Q4: 对话会保存吗？
**答**：对话仅在当前会话中保存，刷新页面会清空。

### Q5: 能帮我直接操作吗？
**答**：AI助手只提供指导，不能直接操作系统。
