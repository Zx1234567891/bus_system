# 公交安全管理系统

## 🔐 登录与注册

系统已集成Google风格登录页面，支持以下演示账号：

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 系统管理员 |
| captain | captain123 | 车队队长 |
| driver | driver123 | 司机 |

**数据库员工登录：** 使用员工工号或姓名作为用户名，默认密码：`123456`

**新用户注册：** 点击登录页面的"创建账号"按钮进行注册，密码会安全加密存储

**初次使用需导入用户凭证表：**
```bash
mysql -u root -proot < user_credentials.sql
```

## 🔒 安全功能

- ✅ 密码SHA-256加密存储
- ✅ 登录失败锁定（5次失败锁定30分钟）
- ✅ 管理员用户管理功能
- ✅ 密码重置和修改
- ✅ 账号激活/禁用

## 👥 用户管理（仅admin）

管理员登录后可在"用户管理"菜单中：
- 查看所有用户信息
- 编辑用户角色和状态
- 重置用户密码
- 解锁被锁定的账号
- 删除用户（不能删除自己）

## 公交安全管理系统

基于Flask的Web应用程序，用于管理公交车队、司机和违章记录。

## 🚀 快速启动

### 方法一：使用主菜单（推荐）
双击 `menu.bat` 打开管理菜单，提供以下功能：
- ✅ 启动/停止/重启系统
- 📊 实时查看系统状态
- 💾 数据库管理（备份、导入、清理）
- 🌐 快速打开浏览器

### 方法二：直接启动
双击 `start.bat` 即可启动系统

## 📋 批处理文件说明

| 文件名 | 功能说明 |
|--------|----------|
| `menu.bat` | 主管理菜单，整合所有功能 |
| `start.bat` | 启动系统服务 |
| `stop.bat` | 停止系统服务 |
| `restart.bat` | 重启系统服务 |
| `status.bat` | 实时监控系统状态 |

## 快速启动

### 方法1：使用启动脚本（推荐）
双击 `start.bat` 文件即可启动项目

### 方法2：手动启动
```bash
# 1. 确保MySQL服务运行
net start MySQL80

# 2. 启动Flask应用
python app.py
```

## 访问系统

- **本地访问**: http://127.0.0.1:5000
- **局域网访问**: http://10.198.21.124:5000

## 停止服务

- **停止Flask**: 按 `Ctrl + C` 或双击 `stop.bat`
- **停止MySQL**: `net stop MySQL80` (需管理员权限)

## 数据库配置

```python
host: localhost
port: 3306
user: root
password: root
database: bus_safety_system
```

配置文件位置：
- `config.py`
- `app.py` (第13-22行)

## 系统要求

- Python 3.7+
- MySQL 8.0
- 依赖包见 `requirements.txt`

## 功能模块

1. 录入司机信息（自动创建用户账号）
2. 录入车辆信息
3. 录入违章信息（**支持视频上传**）
4. 查询车队司机
5. 查询司机违章详情（**支持视频查看**）
6. 查询车队违章统计

## 🎥 新增功能

### 违章视频上传与查看

- ✅ **上传视频证据**：录入违章时可上传视频文件（最大100MB）
- ✅ **支持格式**：MP4、AVI、MOV、WMV、FLV、MKV、WebM
- ✅ **实时预览**：上传后即可预览视频内容
- ✅ **弹窗播放**：查询违章记录时点击按钮在小窗中观看视频
- ✅ **自动管理**：视频自动存储到 `static/uploads/videos/` 目录

**详细说明**：参见 `视频上传功能说明.md`

### 自动账号创建

- ✅ **新增司机自动创建账号**：添加司机时自动创建登录账号
- ✅ **默认密码**：123456（司机可登录后修改）
- ✅ **批量创建**：运行 `create_missing_driver_accounts.py` 为现有司机创建账号
